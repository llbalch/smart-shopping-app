# Installation

## Prerequisites

- Node.js (v16+ recommended)
- npm


## After forking and cloning the repository, run the following in terminal:
```bash
npm install
npm run dev
```
 Open browser to http://localhost:5173

## API Endpoints
POST /api/shopping-lists
PUT /api/shopping-lists
DELETE /api/shopping-lists
